# Task 3: Password Generator

## Description
This project is a Password Generator application developed using Python.
It generates strong and random passwords based on user-defined length.

## Features
- User-defined password length
- Random and secure password generation
- Uses a mix of letters, numbers, and symbols

## Concepts Used
- Random module
- Strings
- Loops
- User input

## How to Run
1. Run the Python file
2. Enter the desired password length
3. The generated password will be displayed on the screen

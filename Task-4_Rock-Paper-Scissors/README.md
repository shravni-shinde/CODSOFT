# Task 4: Rock-Paper-Scissors Game

## Description
This project is a simple Rock-Paper-Scissors game developed using Python.
The user plays against the computer, and the winner is decided based on game rules.

## Features
- User vs Computer gameplay
- Random computer choice
- Displays winner of each round
- Option to play multiple rounds

## Game Rules
- Rock beats Scissors
- Scissors beats Paper
- Paper beats Rock

## Concepts Used
- Random module
- Conditional statements
- Loops
- User input

## How to Run
1. Run the Python file
2. Choose Rock, Paper, or Scissors
3. View the result and play again if desired

# Task 1: To-Do List Application

## Description
This project is a To-Do List application developed using Python.
It allows users to create, update, and manage daily tasks efficiently
through a command-line interface.

## Features
- Add new tasks
- View all tasks
- Update existing tasks
- Delete completed tasks

## Concepts Used
- Lists
- Loops
- Conditional statements
- User input

## How to Run
1. Run the Python file
2. Choose an option from the menu
3. Manage your tasks easily
